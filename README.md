# HabitBloom 🌱

## Project Description

HabitBloom is a simple and interactive web application designed to help users build and maintain positive daily habits. The application allows users to add, track, and manage habits efficiently through an easy-to-use interface.

The goal of this project is to promote productivity and consistency by enabling users to monitor their daily routines and stay motivated.

---

## Features

* Add new habits to track daily activities
* View a list of all added habits
* Mark habits as completed
* Delete habits when no longer needed
* Clean and responsive user interface

---

## Technologies Used

* **HTML** – Structure of the web pages
* **CSS** – Styling and layout design
* **JavaScript** – Functionality and user interaction
